# Customize Cluster Number

Customize the number of cluster in the Seurat object

## Usage

``` r
FixedNumClusters(seu, n_clusters, verbose = FALSE)
```

## Arguments

- seu:

  A `Seurat` object

- n_clusters:

  A scalar, the desired number of clusters

- verbose:

  Default is [`FALSE`](https://rdrr.io/r/base/logical.html)
